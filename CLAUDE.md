## Architecture

This is a **Remotion** video project — videos are authored as React components and rendered to MP4. Each video lives in `src/<VideoName>/index.tsx` and exports a `React.FC` (no props needed for TTS-driven videos).

### Composition Registry (`src/Root.tsx`)

All compositions are defined in a single `compositions` array of `CompConfig` objects. To add a new video, insert one entry:

```typescript
{
  id: "MyVideo",
  component: MyVideo,
  durationInFrames: 1500,
  orientation: "portrait",  // "portrait" | "landscape"
  category: "video",        // "video" | "demo"
  description: "...",
}
```

Portrait = 1080×1920, Landscape = 1920×1080. All at 30fps by default (override with `fps`). Resolution is auto-derived from orientation.

### Video Component Patterns

Two patterns are used across the project:

1. **TTS-driven (most common):** Audio from `public/<project>/`. Scene transitions via `TransitionSeries` + `linearTiming`. Subtitles via `SubtitleBar` from shared components. Each scene is a sub-component with hardcoded frame ranges (see `PerformanceMetrics`, `GPTEvolution`).

2. **Simple/animation-driven:** No audio/subtitles, just animated React elements. `ErisPet` plays a GIF; `HelloWorld` uses `Sequence` + `spring`.

Key Remotion APIs used: `useCurrentFrame`, `useVideoConfig`, `interpolate`, `spring`, `AbsoluteFill`, `staticFile`, `Sequence`, `TransitionSeries`.

### Shared Components (`src/shared/components/index.tsx`)

- **FadeIn** — opacity + translateY on frame delay
- **SlideUp** — stronger vertical entry animation
- **WhiteCard** — standard card with border/shadow
- **SubtitleBar** — bottom-positioned subtitle bar with fade in/out (takes `text`, `startFrame`, `endFrame`)
- **SHARED_COLORS** — universal color palette

### Static Assets

`public/` is organized by project subdirectory. Use `staticFile("project-name/filename.ext")` to reference assets (Remotion resolves these at render time). Audio files are typically TTS-generated by Python scripts in `scripts/`.

## Configuration

- `remotion.config.ts` — enables Tailwind v4, sets JPEG image format, overwrite output
- `eslint.config.mjs` — extends `@remotion/eslint-config-flat`
- `tsconfig.json` — ES2018 target, CommonJS modules, strict mode, `noUnusedLocals: true`
- `.prettierrc` — 2-space indent, no tabs, bracket spacing

## TTS Pipeline

Python scripts in `scripts/` generate audio files from text. Each script project has its own directory. The generated audio goes into `public/<project>/`. Subtitle text in video components should match TTS output exactly — it's captured in `SUBTITLE_TEXTS` objects used with `SubtitleBar`.

## Video Composition Lifespan

Each composition's total frame count must account for scene durations plus transition overlaps. `TransitionSeries` subtracts transition frames from adjacent sequences, so:
```
totalFrames = sum(sceneDurations) - sum(transitionFrames)
```
See `PerformanceMetrics/index.tsx` for a worked example with comments.
